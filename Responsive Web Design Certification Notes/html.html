See just like that! You can also link to internal / on page things. This allows users to jump around a page taking them back to the top of the page or down to the main content from the top. This is done with "#id_to_go_too" as the href attribute. If you don't add a an id-to-go_too and instead have just href="#" it acts as a placeholder link. This can be useful for development when you know you need a link but might know know the exact url.

Jump to Bottom will NOT work on this page because there is no footer id for it to find.

You can also add targets to links to tell the browser where to open the link. For example target="_blank" will open a link in a new window/tab.
