Responsive Web Design

Today we access the web via small phone screens and large 82 inch TV's. To ake sites look nice at all screen sizes you can change css based on screen sizes.

Media Queries

You can declare a media query with [ @media () {styles} ]. In the () you specify the value you want to check. The values can be width/height of viewport (screen size), orientations, and resolution.

Ex:
@media screen and (max-width: 600px) {
div.example {
display: none;
}
}

In the above example the styles are applied to screens 600px and LESS.

Use responsive units for width and sizes like view port units!
